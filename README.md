# Student Management System

A command-line Student Management System built using Python, Object-Oriented Programming (OOP), and File Handling.

# Features

- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Automatic File Saving
- Automatic File Loading
- Duplicate Roll Number Validation

# Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
- File Handling
